## 과제 요약
         기존 MP3에서 날짜 및 현재시간과 그림을 추가함으로써, 편의성을 확대하였습니다.
## 과제에 필요한 기술
- **날짜 및 현재시간 추가** 
         현재의 시간을 파악할수 있도록 시간을 추가합니다.
- **그림 추가**
         MP3속 시각적 효과를 주어서 공감각적 효과를 증대시켰습니다.
